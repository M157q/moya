<rows> a collection of rows in a table

name
rows
xml namespace
http://moyaproject.com/tables

Defines a collection of rows in a <table>.

A <rows> tag may contain either <row> tags or – if the src attribute is set – <cell> tags.

When the src attribute is set, it should be a sequence of values to be iterated over (like <for>). Each item in the sequence will generate a row containing the enclosed <cell> tags.
