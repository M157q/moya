Tags used to create table content.

Tag Synopsis
<cell> a cell in a table
<columns> define the columns in a table
<header> define a column header in a table
<row> define a row in a table
<rows> a collection of rows in a table
<sort-header> define a sortable header column in a table
<table> add a table to content
