<parameter> define an rpc parameter

name
parameter
xml namespace
http://moyaproject.com/jsonrpc

Defines a parameter in an RPC call.

Example

<rpc:error code="100" if="format not in ['short', 'medium', 'long', 'full']">
    Format parameter is not correct
</rpc:error>
attributes
name purpose type required? default choices
default Default value expression No None
name Name of the parameter text Yes
required Required? boolean No yes
type Parameter type (number, string, bool, list, object, anything) text No "anything"
